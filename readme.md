# monty
Shopping planner (express, mongodb, react + redux).

Download all needed libs (node.js and yarn required):
```bash
# from backend/ and frontend/ dirs:
$ yarn
```

Run server:
```bash
# from backend/ dir:
$ yarn start
```

Build and watch (or just build) frontend:
```bash
# from frontend/ dir:
$ yarn start-frontend

# or (for just build):
$ yarn build-frontend
```
